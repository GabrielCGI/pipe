# RAT Manager README

Finds all texture files in a Houdini material node, checks if they have a corresponding .rat file, and allows you to convert them if needed. 

## How to use 

Select a Houdini material node (materiallibrary), open the .rat manager tool in the houdini toolbox (Illogic shading -> .rat manager). 
Here, all the texture files should be displayed, you can click convert all to create a corresponding .rat file for all missing files
