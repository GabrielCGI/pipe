chaque fichier a sa propre shelf

- illogic_Lighting  gère tous les outils lighting
- illogic_Shading   gère tous les outils Shading
- illogic_FxCfx     gère tous les outils FX et de CFX

et illogic_solaris8 va gérer les outils générals


Le shelfSet est géré dans le dossier illogic_solaris8. C'est ici qu'on va pouvoir choisir ce qu'on veut ajouter comme shelf dans le shelfSet. C'est lui qui va permettre de rendre actifs les différentes shelfs que l'on veut avoir aux démarrages de Houdini. Comme avoir la shelf de prism / celle des lop / celle de sideFx Labs etc..
